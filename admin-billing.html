إعداد Stripe
خطوات تفعيل Stripe:
1. اذهب إلى dashboard.stripe.com
2. سجّل حسابك من أي بلد (الدنمارك DK، ألمانيا DE، السعودية SA...)
3. اذهب إلى: Developers → API Keys
4. انسخ Publishable Key (يبدأ بـ pk_live_ أو pk_test_)
5. افتح ملف billing.html
6. ابحث عن: STRIPE_PUBLISHABLE_KEY: ''
7. ضع مفتاحك بين الأقواس: STRIPE_PUBLISHABLE_KEY: 'pk_live_xxxxx'

⚠️ هذا الحقل للحفظ المرجعي فقط — المفتاح الفعلي يجب وضعه في billing.html

إعداد PayPal
خطوات تفعيل PayPal:
1. اذهب إلى developer.paypal.com
2. My Apps & Credentials → Create App
3. انسخ Client ID
4. افتح billing.html وابحث عن: PAYPAL_CLIENT_ID: ''
5. ضع Client ID بين الأقواس
